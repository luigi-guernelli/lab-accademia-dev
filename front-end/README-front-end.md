# front-end
------------------------------------------------------------------------------------------------------------------
Il front-end è la parte visibile di un'applicazione o di un sito web, con cui l'utente interagisce direttamente.
Tecnologie principali: HTML, CSS, JavaScript (e framework come React, Angular o Vue.js).
Responsabilità: Grafica, layout, animazioni, gestione dell’input utente e comunicazione con il back-end.
------------------------------------------------------------------------------------------------------------------
* Front-end: Il design di una pagina di login (input, pulsanti, animazioni).
