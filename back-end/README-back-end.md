# back-end
> Il back-end è la parte "nascosta" che gestisce la logica, i dati e le operazioni lato server.
> Tecnologie principali: Linguaggi come Python, Java, PHP, Node.js e database come MySQL, MongoDB.
> Responsabilità: Elaborazione delle richieste, gestione del database, autenticazione e sicurezza.
----------------------------------------------------------------------------------------------------
* Back-end: Verifica delle credenziali e accesso al database per controllare l’utente.
