# Array
- Un array è una struttura fissa che può contenere più valori dello
 stesso tipo es: (int, String, ecc).
- La lunghezza è fissata al momento della creazione e non può cambiare.
------------------------------------------------------------------------------
# Sintassi (java)
// Dichiarazione e inizializzazione
int [] numeri = new int[5]; // Array di 5 interi
String[] nomi = {"Alice", "Bob", "Charlie"}; // Array pieno
------------------------------------------------------------------------------
- <b>Accesso agli elementi: </b> Si usa un indice (partendo da 0);
System.out.println(nomi[0]); // Stampa "Alice"
