# Cicli (Loop)
Strutture di controllo che ripetono un blocco di codice fino a quando una condizione è vera.
--------------------------------------------------------------------------------------------------
# Esempi comuni:

<b>for:</b> Ripete un blocco un numero predefinito di volte.

<b>while:</b> Ripete finché una condizione è vera.

<b>do-while:</b> Esegue il blocco almeno una volta, poi verifica la condizione.
--------------------------------------------------------------------------------------------------
# Istruzioni condizionali
Strutture che eseguono un blocco di codice solo se una condizione è vera.
--------------------------------------------------------------------------------------------------
# Esempi comuni:

<b>if:</b> Esegue codice se la condizione è vera.

<b>else:</b> Specifica un'alternativa se la condizione è falsa.

<b>else if:</b> Verifica condizioni multiple in sequenza.

<b>switch:</b> Seleziona un blocco in base al valore di una variabile.
--------------------------------------------------------------------------------------------------
# Differenza chiave:

- I cicli ripetono azioni.

- Le condizioni decidono se eseguire un'azione.
